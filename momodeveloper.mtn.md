# momodeveloper.mtn.com

> MTN's Mobile Money Developer Portal offers resources for developers to integrate and utilize MTN's mobile financial services. The portal features detailed product information, API documentation, FAQs, and tools for account and password management, catering to various countries in Africa.

## URLs

- [apis](/apis): Overview of available APIs for integration with MTN services.
- [api-details](/api-details): In-depth details on specific APIs available to developers.
- [sandboxcase](/sandboxcase): Sandbox environment for testing integrations.
- [Product-descriptions](/Product-descriptions): Descriptions of various products offered under MTN's mobile money services.
- [faqs](/faqs): Frequently asked questions covering common inquiries about the services.
- [api-documentation/api-description](/api-documentation/api-description): Documentation and descriptions of specific APIs.
- [faq](/faq): Another section for frequently asked questions.
- [api-documentation/brand-guidelines](/api-documentation/brand-guidelines): Guidelines for the use of MTN's brand and related assets in integrations.
- [reports](/reports): Page for accessing various reports related to service usage.
- [widget-api](/widget-api): Information regarding APIs specifically for widget integration.
